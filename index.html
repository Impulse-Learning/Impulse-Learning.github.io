Web Design Class

All successful modern businesses have a good website. For companies like Facebook, Craigslist, and more, their website is the bulk of the company. Even for small businesses, a good website is critical to success. In this class, you will learn the basics of web design using HTML and CSS. Students will walk out of this class with the skills they need to build a simple webpage and the ability to apply it to their own projects down the road. No previous programming experience is required for this class.

Python Programming

Are you interested in coding, computers, and other technology? This class goes over the basics of programming using the Python language. We cover variables, loops, conditional statements, methods, and more while also diving deeper into specific Python libraries. You will finish this class having completed 3 fun and interesting projects using different aspects of your new programming knowledge! No prior experience programming is needed for this course.

Model United Nations

Model United Nations (MUN) helps students hone their leadership, research, writing, public speaking, and problem solving-skills. This course prepares students to succeed by introducing them to parliamentary procedure, impromptu speaking, and position paper structure. By the end of the course, students will participate in an Impulse Learning Model United Nations Conference.
